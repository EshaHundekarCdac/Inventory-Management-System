# Inventory-Management-System
This is a Python-based Inventory Management System built using Flask, MySQL, and Pandas. It allows users to manage products, track categories (Electronics, Perishables, etc.), and visualise inventory data.

## Features
- Product Management: Add, delete, and update product quantities and prices.
- Polymorphism: Different product categories (like Electronics) have unique logic for calculating total value.
- Analytics: Visualize inventory data using Matplotlib bar charts, pie charts, or histograms.
- Exporting: Export current inventory reports to CSV using Pandas.

## Prerequisites
Ensure you have the following installed:
- Python 3.x
- MySQL Server
- `pip` (Python package installer)
